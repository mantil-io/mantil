## mantil env

Show project environment variables

### Synopsis

Show project environment variables

You can set environment variables in terminal with:
$ eval $(mantil env)


```
mantil env [flags]
```

### Options

```
  -s, --stage string   stage name
  -u, --url            show only project api url
```
